# IT Support Portfolio

This repository showcases hands-on IT Support labs and documnetation demonstrating practical skills in troubleshooting, user account management , access control, and system administration.

The labs in this portfolio are desgned to simulate real-world IT Supoort tasks commonly performed in entry-level help desk and junior systmes administrator roles.

---

## Skills Demonstrated

- User account management (Windows & Linux)
- Password resets and account lockout remediation
- Group management and permissions
- Least-privilege access control
- Command-line administration
- Troubleshooting and verification
- Clear technical documentation
